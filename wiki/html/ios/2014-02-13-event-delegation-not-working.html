ios 事件代理失效 
html:
    <div>
        <span id="span_1">span_1</span>
        <span id="span_2">span_2</span>
        <div id="dv_1">dv_1</div>
        <a href="javascript:;" id="a_1">a_1</a>
    </div>
js:
    document.addEventListener('click', function(e) {
        alert('bind document e.target.id = '+e.target.id);
    },false);

上面的代码在ios下点击 span_1, span_2, dv_1 都没有弹出框。

解决方案：

1, 标签加上 onclick 属性

2, 或者在body下添加一个外层
